# Agent Rules

## Frontend Component Rule (shadcn/ui)
When building or modifying frontend UI in this repository:
- Use shadcn/ui components by default.
- Prefer composing from the approved shadcn component set before creating custom primitives.
- If a needed interaction is not covered by this list, call out the gap explicitly before introducing custom UI primitives.
